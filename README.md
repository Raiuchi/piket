# 🚄 ПИКЕТ

> **Пикет** для контроля ограничений скорости по км/пикетам на Октябрьской железной дороге.

## ✨ Что умеет

| | |
|---|---|
| 🛰 | **Точная позиция по GPS** — снэппинг к реальным координатам пути, без накопления ошибки одометра |
| 🚦 | **Сигнализация на подходе** — зелёный / жёлтый / красный, голос + звук + вибро |
| 📍 | **Калибровка по километровому столбу** — один раз привязался, дальше GPS сам ведёт счёт |
| 🔄 | **Автоопределение перегона** — сам понимает, где ты, по координатам |
| 🌙 | **Ночной режим** — не слепит в кабине |
| 📋 | **Список ограничений** — сортировка от ближайшего к тебе |
| 🔋 | **Работает в фоне** — с погашенным экраном, через foreground-службу |
| 🔔 | **Уведомление с позицией** — км/пк прямо в шторке |
| 📴 | **Полностью офлайн** — все данные хранятся на телефоне |

## 🛠 Технологии

- **Java** + Android `WebView` — нативная обёртка
- **Foreground Service** — фоновая работа, не выгружается системой
- **Geolocation API** + реальные координаты пути для точного позиционирования
- **SpeechSynthesis** — голосовые оповещения
- Полностью **offline-first**, без серверной части

## 📦 Сборка из исходников

Сборка APK происходит автоматически через **GitHub Actions** при каждом пуше:

1. Зайди во вкладку **Actions**
2. Дождись зелёной галочки ✅
3. Скачай артефакт **`piket-apk`**
4. Установи `ПИКЕТ.apk` на телефон
