# ПИКЕТ — контроль ограничений скорости

Нативная Android-обёртка (WebView + фоновая служба) над приложением ПИКЕТ.

## Сборка APK через GitHub (без компьютера)
1. Создай репозиторий на github.com
2. Создай файл `.github/workflows/build.yml` (содержимое — в чате)
3. Загрузи `project.zip` в репозиторий
4. Вкладка **Actions** → дождись сборки → скачай артефакт **piket-apk**
5. Распакуй, установи `app-debug.apk` (разреши установку из неизвестных источников)
