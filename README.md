## Что это

**ПИКЕТ** — инструмент для машинистов скоростных поездов Октябрьской ж/д. Ведёт точный счёт километров и пикетов по GPS, заблаговременно предупреждает о введённых ограничениях скорости: голосом, звуком и визуально. Работает в фоне с погашенным экраном — телефон можно убрать в карман.

> ⚠️ **Помощник, не замена.** Приложение не заменяет АЛСН / КЛУБ-У и действующий регламент.  
> Решения принимаются машинистом на основании показаний штатных приборов безопасности.

## GPS: как работает и как защищён

Приложение использует нативный **Android FusedLocationProvider** — объединяет спутниковый GPS, данные вышек сотовой сети и WiFi для наилучшей точности в любых условиях.

**Многослойная защита от помех и глушения (стресс-тест 27/27 ✅):**

| Слой защиты | Что делает |
|---|---|
| **Потолок 255 км/ч** | Всё выше отбрасывается — физически невозможно для Сапсана |
| **Фильтр скачков 25 км/ч/с** | Отброс резких изменений при нормальной езде (6× запас от торможения) |
| **Насыщение 80 км/ч за фикс** | Защита от забросов после длинных пауз (туннель, глушение) |
| **speedRecovery** | После потери сигнала скорость восстанавливается за 1–2 замера, не через 8 сек |
| **Watchdog пересоздания** | Пересоздаёт GPS-клиент при зависании более 15 секунд |
| **Запасной канал** | При молчании основного >15с включается второй канал через вышки |
| **Drift-подбивка** | При 20+ сек систематического расхождения GPS/счёт — автокорректировка |

## Когда нужен интернет

| | Нужен? |
|---|:---:|
| Основная работа (счёт км/пк, сигнализация, голос) | ❌ |
| GPS по спутникам | ❌ |
| **GPS через вышки сотовой сети** (часть FusedLocation, резервный канал) | ✅ |
| Шрифты интерфейса (Manrope, JetBrains Mono) — загружаются один раз | ✅ |
| Скачивание и обновление APK | ✅ |

> **Про вышки:** FusedLocationProvider на Android автоматически использует базовые станции сотовой сети как дополнительный источник позиции — особенно полезно при первом запуске (A-GPS, быстрое получение первого фикса) и в туннелях. Точность 50–300м против 5–15м у спутника, но для грубой привязки при потере сигнала — лучше, чем ничего.

## Технологии

- **Java** + Android `WebView` — нативная обёртка с Java Foreground Service
- **FusedLocationProviderClient** — объединяет GPS/Network/WiFi в один поток
- **Headless WebView** — JS-логика работает в фоне, независимо от UI
- **Geolocation API** + координаты пути — снэппинг к реальному маршруту
- **SpeechSynthesis** — голосовые оповещения
- **Offline-first** — все данные локально, без серверной части

## Установка

1. Скачай APK по кнопке выше
2. Открой файл на телефоне — Android спросит разрешение на установку из неизвестных источников
3. Разреши, установи — иконка появится на главном экране

Приложение само уведомит о новой версии. При обновлении — просто установи поверх старой, данные сохранятся.
